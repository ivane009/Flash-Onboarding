# Plan: Actualizar APIs en bitcoin-status.html

## Objetivo
Reemplazar las APIs actuales (Coinbase, Mempool.space para bloque) con las APIs especificadas por el usuario para obtener datos reales de Bitcoin.

## APIs a utilizar

| Datos | API | Endpoint |
|-------|-----|----------|
| Precio BTC/USD y BTC/EUR | CoinGecko | `https://api.coingecko.com/api/v3/simple/price?ids=bitcoin&vs_currencies=usd,eur` |
| Bloque actual | Blockchain.info | `https://blockchain.info/latestblock` |
| Mempool (sat/vB) | Mempool.space | `https://mempool.space/api/v1/fees/recommended` |

## Cambios en el código

- Endpoint: `https://api.coingecko.com/api/v3/simple/price?ids=bitcoin&vs_currencies=usd,eur`

- Mantener try/catch independiente para USD y EUR

### 2. fetchNetwork() - Usar Blockchain.info para bloque

- Respuesta: `{ height: number, hash: string, time: number, ... }`
- Extraer `height` para mostrar número de bloque
- Mantener Mempool.space para fees: `https://mempool.space/api/v1/fees/recommended`
