## "Failed to fetch" Error - Diagnosis

The error occurs because the frontend cannot reach the backend. Possible causes:

1. **Backend not running**: Server needs to be running on port 3001
2. **CORS not configured**: Backend may not allow requests from frontend origin
3. **API URL mismatch**: Frontend uses `localhost:3001` but backend may be bound to different interface
4. **Windows PowerShell issue**: `app.listen(PORT)` without IP binding only listens on `localhost`, not `0.0.0.0`

## Steps to Implement EmailJS

### Step 1: Create EmailJS Account
1. Go to https://www.emailjs.com and sign up
2. Verify your email address

### Step 2: Create EmailJS Service
1. In EmailJS dashboard, go to "Email Services"
2. Click "Add New Service"
3. Select your email provider (Gmail, Outlook, or SMTP)
4. Follow setup instructions
5. Copy the **Service ID** (e.g., `service_xxxxx`)

### Step 3: Create Email Template
1. Go to "Email Templates" in EmailJS dashboard
2. Click "Create New Template"
3. Create template for OTP verification:

**Template Example:**
```
Subject: Tu código de verificación Flash: {{otp_code}}

Hola {{user_name}},

Tu código de verificación es: {{otp_code}}

Este código expira en 15 minutos.

¡Gracias por usar Flash!
```

4. Create template for password reset:
```
Subject: Restablecer contraseña - Flash

Hola {{user_name}},

Usa este código para restablecer tu contraseña: {{reset_code}}

Este código expira en 1 hora.

Si no solicitaste este cambio, ignora este email.
```

5. Copy the **Template ID** (e.g., `template_xxxxx`)

### Step 4: Get Your User ID
1. Go to Account > API Authorization
2. Copy your **Public Key** (e.g., `user_xxxxx`)
